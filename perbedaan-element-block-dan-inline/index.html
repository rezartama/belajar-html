BLOCK ELEMENT

block element adalah element atau tag yang mengambil satu baris browser

jadi jika ada tag selanjutnya maka akan dimunculkan dibagian bawah

INLINE ELEMENT

inline element adalah element atau tag yang tidak mengambil satu baris

ini berlawanan dengan block element, maksudnya inline tuh jadi besar atau panjangnya karakter

ya nyesuain aja jadi kayak kalo masih ada space di sampingnya

kita bisa nambahin element inline disebelahnya juga jadi satu baris gitu
